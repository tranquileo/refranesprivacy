Privacy Policy

This extension does not collect, store, or share any personal data.

Local Data Usage

All functionality is handled locally within the user's browser. The extension uses local storage only to:

• Save user preferences (such as theme)
• Track usage count
• Avoid repeating previously shown content

No External Communication

No personal data is transmitted to any external servers. The extension works entirely offline.

No Tracking

This extension does not use tracking technologies, analytics, or third-party services.

No Data Monetization

We do not sell, share, or monetize user data in any form.
